MultiMap<K, V>

  • K

    The type of keys maintained by this map.

  • V

    The type of mapped values.

MultiMap provides a generic way to store multiple values associated with the same key. This class is similar to a MapK, List >, but it provides additional convenience methods to make it easier to work with a list of values for a given key. By default, duplicate values for a key are not allowed, but this can be configured using the acceptDuplication flag in the constructor.
